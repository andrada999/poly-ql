# Grammar
***

### Questionaire
A questionaire consists of a list of questions, which can be arranged:

* sequentially
* conditionally (if-statements)


### Question
A question consists of three components:

* Identifier (*string*)
* Label (*string*)
* Type (*type*)

### Expression
* Boolean
  * &&
  * ||
  * !
* Comparison
  * <
  * >
  * <=
  * >=
  * ==
  * !=
* Arithmetic
 * +
 * -
 * *
 * /

### Type
* bool
* string
* int
* date
* decimal
* money
